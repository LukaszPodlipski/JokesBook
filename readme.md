
# JokesBook

Simple app to share jokes :)

## Run Locally

Clone the project

```bash
  git clone https://github.com/LukaszPodlipski/jokesbook.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node app.js
```
